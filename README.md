# 🎙️ LingoBuddy - AI English Tutor

LingoBuddy is an AI-powered English speaking practice application that helps users improve their spoken English through natural voice conversations.

## ✨ Features

- 🎤 Voice-based English conversation practice
- 🤖 AI-powered responses using Google Gemini
- 📝 Grammar and vocabulary correction tips
- 🔊 Text-to-Speech responses
- 🎨 Modern Glassmorphism UI
- ⚡ Real-time Speech Recognition
- 📱 Responsive Design

## 🚀 Installation

### 1. Clone the Repository


git clone https://github.com/yourusername/lingobuddy.git
cd lingobuddy


### 2. Install Dependencies

npm install


### 3. Create Environment Variables

Create a `.env` file inside the backend folder:

env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY

## 🎯 How It Works

1. Click the microphone button.
2. Speak in English.
3. Speech Recognition converts speech to text.
4. Text is sent to Gemini AI.
5. AI generates:
   - A conversational response
   - Grammar correction suggestions
6. Response is displayed on screen.
7. Text-to-Speech reads the response aloud.



## 💬 Example

### User


I goes to market yesterday.


### AI Correction


Instead of "I goes to market yesterday", say:
"I went to the market yesterday."

### AI Response

Nice! What did you buy at the market?
